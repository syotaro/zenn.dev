## ## よく使う

### Explorerからファイルを開く

`shift + command + e`
Explorerにフォーカスが移動。Enterを押すとファイル名変更。spaceでファイルが開く

### ウィンドウ移動

`control + tab`
開いているタブを移動。

### クイックオープン

`command + p`
Quick Openのウィンドウを開く（一度利用したファイルなどを表示）

`shift + command + p`
「>」を入力した状態でウィンドウが開き、そのままコマンドをインクリメンタルサーチ

### ファイル全文検索

`shift + command + f`
Searchにフォーカスが移動。
ワークスペース内のファイルにgrep検索を実行

## ## よく忘れる

### 矩形選択（くけいせんたく）

`option + command + ↑↓`
複数行に同一の操作をするときに便利。

### 行移動

`option + ↑↓`
カーソル行を上下に移動。

`shift + option + ↑↓`
カーソル行の複製。

### パネル移動

`control + q`
vscode内のいろいろなパネルにフォーカスを移動

### ショートカット一覧

`command + k` → `command + s`
